🏓 Pong Game – Python (OOP)
Bu proje, Python dilinde *Nesne Yönelimli Programlama (OOP)* yaklaşımı kullanılarak geliştirilmiş basit bir Pong oyunudur.  
Oyun, Turtle grafik modülü ile tasarlanmıştır.

 📁 Proje Dosyaları
- pong.py – Oyunun ana Python dosyası

 🚀 Kurulum
1. Python 3 yüklü değilse [buradan](https://www.python.org/downloads/) indirip kur.
2. Bu projeyi .zip olarak indir ya da terminalden klonla:
   ```bash
   git clone https://github.com/kullaniciadın/pong-game.git
3. pong.py dosyasını çalıştır:
python pong.py



🎮 Nasıl Oynanır?
Sol Oyuncu: W ve S tuşları ile yukarı–aşağı hareket eder.
Sağ Oyuncu: Yukarı ve Aşağı ok tuşları ile oynar.
Top iki oyuncudan biri kaçırdığında karşı taraf puan kazanır.
Oyunu başlatmak için Enter’a basılır.
